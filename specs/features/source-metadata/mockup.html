Source metadata — side panel mockup

Try it: click a confidence badge on the right of any input to set low/medium/high, click a source line to open the source+comment editor, or change an input's value — the confidence resets (pulses red) while the comment persists.

In the source editor, "Custom source…" reveals the name + link fields. Typing a name that collides with a canonical source (e.g. ADEME study — Base Empreinte) shows the quiet notice described in §4.

Standalone mockup — all interactions are client-side only.
